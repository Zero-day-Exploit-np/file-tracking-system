# Code of Conduct

## Our Commitment

We are committed to providing a welcoming, respectful, and inclusive environment for everyone participating in this project, regardless of age, experience level, gender identity, nationality, ethnicity, religion, disability, or background.

We aim to foster a community where everyone feels safe to collaborate, learn, and contribute.

---

## Expected Behavior

Contributors are expected to:

- Be respectful and courteous to others.
- Use welcoming and inclusive language.
- Accept constructive feedback gracefully.
- Focus on what is best for the project.
- Help create a positive environment for everyone.
- Respect different viewpoints and experiences.
- Report bugs and security issues responsibly.

---

## Unacceptable Behavior

The following behaviors are not tolerated:

- Harassment or discrimination of any kind.
- Personal attacks, insults, or offensive comments.
- Trolling or intentionally disruptive behavior.
- Publishing private or confidential information without permission.
- Spamming or intentionally submitting malicious code.
- Any behavior that creates an unsafe or hostile environment.

---

## Contribution Standards

When contributing, please:

- Follow the project's coding standards.
- Write clean, readable, and maintainable code.
- Test your changes before submitting.
- Keep pull requests focused on a single feature or fix.
- Update documentation when necessary.

---

## Reporting Issues

If you experience or witness unacceptable behavior, please report it to the project maintainers.

When reporting an issue, include:

- A description of what happened.
- Relevant screenshots or logs (if applicable).
- Steps to reproduce the issue (if applicable).

All reports will be reviewed confidentially and handled appropriately.

---

## Enforcement

Project maintainers are responsible for enforcing this Code of Conduct.

They may take appropriate action, including:

- Warning contributors.
- Removing inappropriate content.
- Rejecting pull requests.
- Temporarily restricting participation.
- Permanently banning users who repeatedly violate these guidelines.

---

## Scope

This Code of Conduct applies to:

- GitHub repositories
- Issues
- Pull Requests
- Discussions
- Project documentation
- Community communication related to this project
